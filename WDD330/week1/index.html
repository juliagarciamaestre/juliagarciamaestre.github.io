Tips to make Mobile Advertising More User Friendly:

  1. Made the advertisment small and simple
  2. If you want to have a large ad, you can made it small but when the user click on it reveals the original size
  3. Video ads generally resut in higher conversion rates
  4. The little "window" ad, it a good idea. When the user scroll the page this "little window" shows the app
  5. Emphaty is super important.
  6. Keep on mind the Reachability of the user.

    For example, on instagram, if we want the user to interact with our content is better to put the forms in the part of the screen that is easy to the user to touch
